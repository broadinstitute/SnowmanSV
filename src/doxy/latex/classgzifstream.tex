\subsection{Detailed Description}
Gzipped file input stream class. 

This class implements ifstream for gzipped files. Seeking and putback is not supported yet.

Stream will be in state good() if file opens successfully; otherwise in state fail(). This differs from the behavior of ifstream, which never sets the state to good() and therefore won't allow you to reuse the stream for a second file unless you manually clear() the state.
