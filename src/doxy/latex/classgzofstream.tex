\subsection{Detailed Description}
Gzipped file output stream class. 

This class implements ofstream for gzipped files. Seeking and putback is not supported yet.

Stream will be in state good() if file opens successfully; otherwise in state fail(). This differs from the behavior of ofstream, which never sets the state to good() and therefore won't allow you to reuse the stream for a second file unless you manually clear() the state.
